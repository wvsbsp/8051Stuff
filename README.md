# 8051Stuff
Eine Sammlung kleiner Projekte für den AT89C51
============================================================================
Compiler RIDE7

- servo8: Servo-Bibliothek für 8 Servos an einem Port
- Foto-Timer: nimmt ab eine Anfangsuhrzeit alle 5min eine serie Bilder auf
  Benutzt den Fernauslöser-Anschuss einer CANON EOS1000D (2.5mm Klinke)
  (entstanden für die Mondfinsternis am 27.9.2015)


A collection of small microcontroller-projects für the Atmel AT89C51
============================================================================
Compiler RIDE7

- servo8: use 8 servos on one port, using only one timer
- foto-timer: take a number of pictures with your CANON DSLR, remote connection
  via the 2.5mm connector.
